## Syntax / Symbols

- `$` atoms
- `()` group expressions are expression / key-value pairs
- `[]` list of values
- `{}` statements
- `@` keyword
- `'T` Generic indentifier
- `` `backtick` `` backticks for template strings
- `_` throw away
- `;` end of statement or expression
- `.` field access OR namespace access
- `,` separate lists
