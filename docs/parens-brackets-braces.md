## Parens, Braces and Brackets

| Bracket | Description                                        |
| --- | ------------------------------------------------------ |
| [ ] | encloses type parameters or type arguments             |
| ( ) | groups expressions, parameter/arugment lists or tuples |
| { } | sequence of statements or definitions                  |

## Literals

The `$` sigil is used for literal brackets that are used for grouping instead of syntax.

`{}` code block (group expressions / statements)
`${}` object literal
`[]` generic type parameters (group parameters)
`$[]` array literal
`()` group expressions
`$()` tuple literal

## Map literal

`[name:type=value,name2:type2=value2]`
