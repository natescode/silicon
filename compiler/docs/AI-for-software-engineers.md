# Contain the Vibes

If you're going to use AI to generate code, you have two choices: generate the tests, or generate the implementation; never both! 

The most important steps in software are defining the requirements, and verification. I recommend having AI generate the tests first, obviously prompting for which types of use cases to cover and having an already defined API. Then using AI Tab completion to help write the actual implementation.
